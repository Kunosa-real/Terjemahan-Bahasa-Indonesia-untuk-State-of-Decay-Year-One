State of Decay YOSE - Indonesian Translation Pack

File:
  zzzz_indonesian_translation.pak

Install:
1. Copy the .pak file into:
   C:\Program Files (x86)\Steam\steamapps\common\State of Decay YOSE\Game\
2. Launch the game (English language setting).
3. To uninstall, remove the .pak file.

Notes:
- This pack does NOT overwrite original game files.
- It overrides language assets from gamedata.pak.
