## 🛠️ Cara Install

> Mudah! Cukup copy satu file saja.

### Langkah 1 — Download
Klik tombol download di atas, pilih versi terbaru

### Langkah 2 — Ambil file mod
Buka folder hasil download, ambil file:
```
build\id_mod\zzzz_indonesian_translation.pak
```

### Langkah 3 — Copy ke folder game
Copy file `.pak` tadi ke folder game:
```
C:\Program Files (x86)\Steam\steamapps\common\State of Decay YOSE\Game\
```

### Langkah 4 — Set bahasa di Steam
1. Buka **Steam** → klik kanan **State of Decay YOSE**
2. Pilih **Properties**
3. Tab **Language** → pilih **English**
4. Tunggu hingga selesai

### Langkah 5 — Jalankan game
Buka game — teks sudah dalam Bahasa Indonesia! ✅

## ❓ FAQ

**Q: Apakah mod ini aman?**
A: Ya, 100% aman. Hanya mengubah teks tampilan, tidak mengubah gameplay.

**Q: Kenapa harus set bahasa ke English di Steam?**
A: Mod ini bekerja di atas file bahasa Inggris. Kalau diset bahasa lain, mod tidak akan terbaca.

**Q: Game saya crash setelah install, kenapa?**
A: Pastikan file `.pak` ditaruh di folder yang benar. Coba verifikasi file game di Steam jika perlu.

**Q: Apakah mod ini gratis?**
A: Ya, selamanya gratis! Tapi donasi sangat diapresiasi 💖
